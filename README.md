# VAE（变分自编码器）
使用Numpy实现VAE模型，并在MNIST数据集上进行样本生成。

## 主要思路
VAE主要优化的是从正太分布中生成的数据与真实数据的损失（二分交叉熵损失或者均方差损失）以及预测分布与真实分布之间的损失（KL散度），模型需要自动训练并权衡这两方面的损失。

## 实验过程
